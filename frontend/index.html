How to use our app

  • Unregistered user
  • Registered user
  • Administrator

Users who are not registered will only be able to view the available categories and the plans they contain.

The registered user, like the anonymous user, are able to view the categories and the plans that contains them. In addition, they can edit their profile data except email, which will be unchangeable once the user register it; mark a plan as completed, view their profile information both on the web and in PDF format if they want to, choose favorite category(ies) and observe the progress for each category, represented in multiple charts which will be explained in the section Charts. Finally, they will receive an email when a tree in a category achieve a specific height.

This user is allowed, in addition to the functionalities described for the previous type of user, to control the record of plans made by users marking them as incompleted, and adding / editing categories and plans.
